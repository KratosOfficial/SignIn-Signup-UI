<h3 align="left">About 📖</h3>

The UI Has 3 Screen OnBoarding, SignIn & SignUp. It is made using Flutter 💙.<br>
The Best part of this UI design is that it has both Login with Email & Phone In One Screen, isn't that awesome.<br>
This UI Design has the form validation too.

<h1 align="center">How to Use 🤔</h1>

````
- Clone this repository:
$ git clone https://github.com/KratosOfficial/SignIn-Signup-UI

- Enter in directory:
$ cd Login

- For install dependencies:
$ flutter packages get

- Run the app: 
$ flutter run

````
